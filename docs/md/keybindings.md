# Keybindings

## Navigation

| Key | Action |
|-----|--------|
| `j` / `↓` | Line down |
| `k` / `↑` | Line up |
| `g g` | Top of buffer |
| `G` | Bottom of buffer |

## Tree Navigation

| Key | Action |
|-----|--------|
| `h` / `←` | Collapse subtree, or jump to parent |
| `l` / `→` | Expand + jump to first child |
| `[` / `Ctrl+↑` | Previous sibling |
| `]` / `Ctrl+↓` | Next sibling |
| `{` | First sibling |
| `}` | Last sibling |
| `u` / `Ctrl+←` | Jump to parent |

## Editing

| Key | Action |
|-----|--------|
| `Enter` | Edit heading title |
| `Esc` | Cancel edit |
| `o` | New sibling below |
| `O` | New sibling above |
| `d d` | Kill heading + subtree |
| `<` / `>` | Unindent / indent |

## States, Priority & Severity

| Key | Action |
|-----|--------|
| `t` / `Space` | Cycle TODO state |
| `p` | Cycle priority (P0 → P1 → P2 → P3) |
| `s` | Cycle severity (S0 → S1 → S2 → S3) |
| `:` | Edit tags |
| `H` | Toggle hide DONE items |

## View

| Key | Action |
|-----|--------|
| `f` / `F` | Cycle global fold |
| `w` | Toggle word wrap |
| `M` | Toggle markdown panel |
| `I` | Toggle metadata annotations |
| `T` | Cycle tab mode (off → by tag → by heading) |
| `⌘S` / `Ctrl+S` | Sync to Google Drive |
| `?` | Toggle help screen |

## Search & Filter

| Key | Action |
|-----|--------|
| `/` / `⌘K` | Fuzzy search |
| `\` | Open / cycle filter group |
| `c` (filter focused) | Clear filters |

## Undo / Redo

| Key | Action |
|-----|--------|
| `z` | Undo |
| `Z` | Redo |
